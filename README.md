# Kaneki Bot 🛡️

**Este bot foi configurado como uma ferramenta de uso final.** O código-fonte principal foi protegido e ofuscado para garantir a segurança do usuário e impedir modificações maliciosas por terceiros.

---

## 🔒 Segurança e Proteção

- **Código Protegido:** Arquivos principais ofuscados para impedir engenharia reversa e edições não autorizadas.
- **Execução Segura:** Comandos de risco como Shell (`$`) e Eval (`>>`) foram removidos para evitar ataques de execução remota de código.
- **Proteção de Dados:** Configuração de `.gitignore` aprimorada para garantir que seus tokens e sessões nunca sejam expostos publicamente.

---

## 🚀 Instalação

Clone o repositório:

```bash
git clone https://github.com/polarminhapomba-crypto/kaneki-bot.git
```

Entre no diretório do projeto:

```bash
cd kaneki-bot
```

Configure e instale as dependências:

```bash
npm run config
npm run config:install
```

---

## ⚡ Iniciar o Bot

Para ligar o bot e começar a usar, execute:

```bash
npm start
```

---

## ⚠️ Aviso Importante

Este repositório é destinado apenas para **instalação e uso**. Qualquer tentativa de modificar os arquivos ofuscados pode corromper o funcionamento do bot e invalidar as proteções de segurança integradas.
