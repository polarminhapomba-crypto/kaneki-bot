# Kaneki Bot

Para instalar e rodar o bot, execute os comandos abaixo, um de cada vez:

1. Clone o repositório:
```bash
git clone https://github.com/polarminhapomba-crypto/kaneki-bot.git
```

2. Entre na pasta do projeto:
```bash
cd kaneki-bot
```

3. Configure o bot:
```bash
npm run config
```

4. Instale as dependências:
```bash
npm run config:install
```

5. Inicie o bot:
```bash
npm start
```
